Hello. It`s a react application to manage contacts.

There are:

1. Adding contact;
2. Deleting contact;
3. Adding tags;
4. Getting all contacts;
5. Adaptive layout;
6. Redux server state;
7. Using Tailwind CSS.

Used technologies:

1. HTML;
2. CSS;
3. JavaScript;
4. React;
5. React-router-dom;
6. Redux;
7. RTK Query;
8. Tailwind CSS;
9. Svgr;
10. Formik.

Link to the live page: https://manage-contacts-app.netlify.app/
